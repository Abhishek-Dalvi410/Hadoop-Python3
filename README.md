# Hadoop-Python3

Hadoop with python3:

Dataset:- Gutenberg

1) Word Count
2) 10 frequent occuring trigrams in Gutenberg
3) Inverted Index
4) Relational Join of two tables
5) K- Nearest Neighbors 

Tasks were run using multiple mappers and reducers hence data was handled accordingly.
